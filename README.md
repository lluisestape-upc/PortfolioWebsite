# Portfolio & Design Showcase

This repository hosts the source code and design architecture for my professional portfolio. The project demonstrates a seamless integration between high-fidelity UI design and modern front-end implementation.

## Overview
The objective of this project was to create a minimalist digital space that highlights my work and expertise. It serves as both a functional website and a case study in component-based web design.

## Tech Stack & Design Standards
* **Architecture:** Component-driven development using **React (JSX)**.
* **Styling:** Utility-first CSS via **Tailwind CSS** for rapid, responsive UI scaling.
* **Typography & Icons:** Optimized for readability and visual consistency using **Lucide** assets.

## Key Assets
* **Interactive Core:** `design-canvas.jsx` contains the primary structural logic and interactive elements.
* **Live Export:** `Portfolio.html` serves as the optimized, production-ready entry point.
* **UX Planning:** `Wireframes.html` documents the initial layout strategy and user flow.
* **Media Assets:** The `/uploads` directory manages optimized imagery for fast load times.
